
# Projeto MKTP Store

O Projeto MKTP Store é um website criado utilizando Next js que gerencia tanto o frontent quanto o backend


## Stack utilizada

**Front-end:** React, Next js, MUI 

**Back-end:** Next js, Next-auth, MongoDB (mongoose), 


## Obs:
